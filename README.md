![](https://raw.githubusercontent.com/dotzero/pad/master/static/images/pad.png)

## Installation using Docker

```
git clone https://github.com/dotzero/pad.git
cd pad
docker-compose up -d
```

Open `http://localhost:8080/` in Browser.
